# Computer Vision & Robotics System

Robotik sistemlerin çevrelerini algılaması ve güvenli bir şekilde hareket etmesi için geliştirilmiş görüntü işleme ve kontrol algoritmasıdır.

## Özellikler
- **Mesafe Tahmini:** Canlı kamera akışından nesnelerin kameraya olan uzaklığının pikseller üzerinden hesaplanması.
- **DWA Algoritması:** Dinamik Pencere Yaklaşımı ile engellerden kaçınma ve yol planlama.
- **Gerçek Zamanlı Karar Mekanizması:** Sensör ve görüntü verilerini birleştirerek kontrol sinyalleri üretme.

## Teknolojiler
- Python
- OpenCV
- NumPy
- Robotik Kontrol Algoritmaları
